## üöÄ Quick Start

### Prerequisites
- Node.js 18+ (for web/backend projects)
- Python 3.9+ (for ML projects)
- npm or yarn
- Git

### General Installation

```bash
# Clone or navigate to project folder
cd <project-folder>

# Install dependencies
npm install

# Start development server
npm run dev
```

### For Mobile Projects (React Native)
```bash
# Install Expo CLI
npm install -g expo-cli

# Start project
expo start
```

### For Flutter Projects
```bash
flutter pub get
flutter run
```
