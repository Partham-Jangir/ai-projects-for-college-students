# Cursor Rules Documentation
## Real-time AI Chatbot for Customer Support

## Project Overview
Real-time AI-powered customer support chatbot with WebSocket communication and intent recognition.

## Tech Stack
- Node.js with Express.js
- Socket.IO for WebSocket
- MongoDB (optional)
- Custom NLP for intent recognition

## Development Guidelines
- Implement WebSocket handlers in separate modules
- Keep intent recognition logic modular
- Handle conversation context
- Optimize for real-time performance
- Test with various user queries

## WebSocket Best Practices
- Use Socket.IO for reliable connections
- Handle connection events properly
- Implement message queuing
- Manage session state
- Handle errors and reconnections

## AI/ML Development
- Document intent patterns
- Handle context in multi-turn conversations
- Optimize intent recognition speed
- Test with edge cases
- Consider sentiment analysis (future)

## Security
- Validate inputs
- Rate limit connections
- Use secure WebSocket
- Protect against attacks
- Encrypt sensitive data
