#  AI-Driven News Aggregator with Personalized Feed

AI-powered news aggregation with personalized content recommendations based on user interests.

## Features
- RSS feed aggregation from multiple sources
- AI-powered content categorization
- Personalized feed based on reading history
- Sentiment analysis on articles
- Trending topics detection

## Tech Stack
- Node.js, Express.js
- Natural Language Processing
- OpenAI for summarization

## Quick Start
```bash
npm install
npm run dev
```

## API Endpoints
- GET /api/news - Get personalized news feed
- POST /api/preferences - Set user preferences
- GET /api/trending - Get trending topics
